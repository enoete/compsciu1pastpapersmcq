TIP Exam Briefing: Common Mistakes
ROM vs RAM: ROM = permanent, RAM = volatile ❌
Cache vs RAM: Cache is faster, RAM is larger
Response time: Higher response time = slower ❌
Registers hierarchy: Registers > Cache > RAM
EPROM vs EEPROM: EPROM uses UV light, EEPROM is electrically erasable
Binary format: write like 10011101 ✔
Two’s complement: 4-bit range = −8 to +7
Keyword traps: watch for NOT, MOST, BEST
